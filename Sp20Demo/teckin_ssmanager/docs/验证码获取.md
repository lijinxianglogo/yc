
### 用户获取验证码
- **接口说明：** 获取验证码发送到用户邮箱，用户注册、更改密码等功能获取
- **接口地址：** /user/getverifycode
- **请求方法：** POST

#### 请求body字段参数

参数名称						|类型		|出现要求	|描述  
:----						|:---		|:------	|:---
Email     					|string		|R			|邮箱
TimeStamp			        |bigint		|R			|当前UNIX时间戳（秒级）

请求示例：

```
{
    "Email":"18203928192@163.com",
    "TimeStamp":1598926576
}
```

#### 返回结果

响应的data字段：

参数名称						|类型		|出现要求	|描述  
:----						|:---		|:------	|:---	
无

响应示例：

```
{
    "Code":200,
    "Msg":"请求成功，验证码已发送",
    "Data":{},
    "MessageId":"9A0A8659F005D6984697E2CA0A9CF",
    "TimeStamp":1598926576
}
```
